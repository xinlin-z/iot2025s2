Project Overview

Smart Cooking is an intelligent system designed to track, monitor, and analyze cooking sessions in real-time using IoT sensors and advanced analytics.

This dashboard provides comprehensive insights into cooking patterns, temperature monitoring, motion detection, and ingredient tracking to help optimize cooking processes and ensure safety.
